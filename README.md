# CaféTrack 📱☕

App móvil para recolección de datos en cafetería (simulación de eventos discretos).

## Cómo publicar en GitHub Pages (gratis)

1. Crea una cuenta en [github.com](https://github.com) si no tienes
2. Crea un repositorio nuevo (público), por ejemplo: `cafetrack`
3. Sube estos 5 archivos al repositorio:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
4. Ve a **Settings → Pages → Source → Deploy from branch → main / root**
5. En ~1 minuto tu app estará en: `https://TU_USUARIO.github.io/cafetrack`

## Instalar en el celular

### Android (Chrome)
- Abre la URL en Chrome
- Toca los 3 puntos → "Añadir a pantalla de inicio"

### iPhone (Safari)
- Abre la URL en Safari
- Toca el botón compartir (cuadro con flecha)
- "Añadir a pantalla de inicio"

La app funciona **sin internet** después de la primera carga (PWA offline).
